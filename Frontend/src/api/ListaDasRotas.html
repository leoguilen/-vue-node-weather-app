ROTAS DA API
Endereço base das rotas: /api/v1
Estados:

  1. Retornar todos estados

    • /states

  2. Retornar estado por id

    • /states/{estado id}

  3. Retornar estado por nome

    • /state/name/{nome estado}
Cidades:

  1. Retornar todas as cidades do estado

    • /cities/state/{estado id}

  2. Retornar cidade do estado por id

    • /cities/byid?stateId={estado id}&cityId={cidade id}

  3. Retornar cidade do estado por nome

    • /cities/byname?stateId={estado id}&cityName={nome cidade}

Clima:

  1. Retornar informações metereológicas

    • /weather/{nome cidade, estado ou país}
